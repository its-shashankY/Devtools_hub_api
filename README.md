# DevTools Hub API

A FastAPI-based REST API for comparing developer tools, tracking pricing, and finding alternatives.

## Features

- **Tools Management**: CRUD operations for developer tools
- **Categories**: Hierarchical categorization of tools
- **Pricing**: Track pricing tiers and changes over time
- **Reviews**: Aggregate reviews from multiple sources
- **Features**: Detailed feature comparison
- **Integrations**: Tool integration network
- **Search & Discovery**: Full-text search and recommendations
- **Analytics**: Pricing trends and tool statistics

## Tech Stack

- **Framework**: FastAPI
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy
- **Migrations**: Alembic
- **Authentication**: JWT
- **Validation**: Pydantic
- **Testing**: Pytest
